# 데이터 사이언티스트 포트폴리오

현대적이고 반응형인 데이터 사이언티스트 포트폴리오 홈페이지입니다.

## 📋 프로젝트 개요

이 포트폴리오 웹사이트는 데이터 사이언티스트의 기술, 프로젝트, 경험을 보여주기 위해 설계되었습니다.

### 주요 특징

- ✨ **현대적 디자인**: 그래디언트 배경과 부드러운 애니메이션
- 📱 **완벽한 반응형**: 모든 기기(데스크톱, 태블릿, 모바일)에 최적화
- 🎨 **전문적 UI**: 데이터 사이언티스트 포지션에 맞는 세련된 디자인
- 🚀 **성능 최적화**: 빠른 로딩 속도와 부드러운 사용자 경험
- 🌙 **다크 모드**: 사용자 편의성을 위한 다크 모드 지원
- ♿ **접근성**: 웹 접근성 표준 준수

## 📁 파일 구조

```
Portfolio/
```

## 🎯 포함된 섹션

### 1. 네비게이션 (Navigation)
- 스티키 네비게이션 바
- 햄버거 메뉴 (모바일 대응)
- 부드러운 스크롤 앵커

### 2. 홈 섹션 (Hero Section)
- 매력적인 소개 문구
- 행동 유도 버튼 (CTA)
- 에니메이션 효과

### 3. 소개 섹션 (About Section)
- 프로필 이미지 플레이스홀더
- 개인 소개 및 전문성 설명
- 통계 정보 (프로젝트 수, 경험, 클라이언트)

### 4. 프로젝트 섹션 (Projects Section)
- 프로젝트 카드 그리드
- 프로젝트별 설명 및 기술 태그
- 반응형 레이아웃

### 5. 기술 스택 섹션 (Skills Section)
- 프로그래밍 언어
- 데이터 분석 및 머신러닝 도구
- 시각화 및 BI 툴
- 도구 및 플랫폼
- 애니메이션된 프로그래스 바

### 6. 연락처 섹션 (Contact Section)
- 연락처 정보 (이메일, 전화, 위치)
- 문의 양식
- 소셜 미디어 링크

### 7. 푸터 (Footer)
- 저작권 정보

## 🔧 기술 스택

### Frontend
- **HTML5**: 시맨틱 마크업
- **CSS3**: 그리드, 플렉스박스, 애니메이션
- **JavaScript**: 바닐라 JS (jQuery 미사용)
- **Font Awesome**: 아이콘 라이브러리

### 라이브러리
- Font Awesome 6.4.0 (아이콘)

## 🚀 시작하기

### 설치 및 실행

1. 저장소 클론 또는 다운로드
```bash
git clone <repository-url>
cd Portfolio
```

2. 로컬 서버에서 실행
```bash
# Python 3 사용
python -m http.server 8000

# 또는 Python 2
python -m SimpleHTTPServer 8000

# 또는 Node.js http-server
npx http-server
```

3. 브라우저에서 열기
```
http://localhost:8000
```

## ✏️ 커스터마이징

### 개인 정보 수정

`index.html`에서 다음 항목을 수정하세요:

- **이름 및 소개**: `hero-title`, `hero-subtitle` 섹션
- **프로필 이미지**: `.profile-placeholder` CSS 수정
- **연락처**: `contact-item` 섹션의 이메일, 전화번호, 위치
- **소셜 미디어 링크**: `social-icon` 링크 수정
- **프로젝트**: `project-card` 섹션 추가/수정
- **기술 스택**: `skill-category` 섹션 수정

### 색상 변경

`style.css`의 `:root` 변수를 수정:

```css
:root {
    --primary-color: #667eea;      /* 원하는 색상으로 변경 */
    --secondary-color: #764ba2;
    --accent-color: #f093fb;
}
```

### 폰트 변경

`style.css`의 `body` 선택자에서 `font-family` 변경

## 📱 반응형 디자인

- **데스크톱**: 1200px 이상
- **태블릿**: 768px ~ 1199px
- **모바일**: 480px ~ 767px
- **소형 모바일**: 479px 이하

## 🎯 주요 JavaScript 기능

1. **햄버거 메뉴**: 모바일에서 네비게이션 토글
2. **스무스 스크롤**: 앵커 클릭 시 부드러운 스크롤
3. **스크롤 애니메이션**: 요소가 뷰포트에 들어올 때 애니메이션
4. **폼 제출**: 연락처 양식 처리
5. **카운터 애니메이션**: 통계 수치 애니메이션
6. **다크 모드**: 라이트/다크 모드 토글
7. **활성 네비게이션 링크**: 스크롤 위치에 따른 네비게이션 하이라이트
